# 🐍 Python Learning Journey

Welcome to my **Python Learning Repository**!  
This repo documents my hands-on journey of learning Python step by step. Instead of just theory, I’m building **mini-projects** and **practical applications** for each topic to truly master the concepts.

---

## 📚 Topics Covered So Far

From basic syntax to advanced concurrency and data validation, here is what is covered:

### 1️⃣ Strings & Basics
- Core Python syntax and environment setup (`01_basics`, `02_virtul-env`).
- **Strings (Immutable Datatypes)**: Properties, slicing, and key methods (`lower()`, `find()`, `split()`, `strip()`, etc.).

### 2️⃣ Mutable Datatypes
- **Lists** 📝: Dynamic arrays with methods like `append()`, `remove()`, `sort()`.
- **Sets** ⭕: Unordered collections with unique elements (`union()`, `intersection()`).
- **Dictionaries** 📖: Key-value pairs for fast lookups.

### 3️⃣ Conditionals (Decision Making)
- Explored `if`, `elif`, `else`, ternary operators, and `match-case`.
- **Mini-Projects:** Smart Kettle System, Snack Suggestion, Smart Thermostat Alert, Delivery Fee Waiver.

### 4️⃣ Loops (Iteration & Flow Control)
- **Constructs:** `for`, `while`, `break`, `continue`, and Python's unique `for-else`.
- **Mini-Projects:** Loop-based mini projects inspired by real-world chai scenarios.

### 5️⃣ Functions
- Basics, built-in functions, parameters, and return statements.
- Explored `global` and `nonlocal` scopes, and various function types.

### 6️⃣ Modules & Packages ☕
- Structured a mock `chai_business` package.
- Learned about `__init__.py`, `main.py`, imports, and organizing helper functions.

### 7️⃣ Comprehensions
- Concise and efficient data generation using **List**, **Set**, **Dictionary**, and **Generator** comprehensions.

### 8️⃣ Generators
- Writing lazy iterators to save memory.
- Concepts covered: `yield`, infinite generators, `send()`, and `close()`.

### 9️⃣ Decorators
- Extending function behaviors elegantly.
- **Mini-Projects:** 
  - 🛠️ Logging Decorator
  - 🔒 Authorization Decorator

### 🔟 Object-Oriented Programming (OOP)
- Mastered classes, objects, namespaces, and constructors.
- **Advanced OOP:** Inheritance, Composition, Method Resolution Order (MRO).
- **Decorators in OOP:** `@staticmethod`, `@classmethod`, and `@property`.

### 1️⃣1️⃣ Exceptions Handling
- Gracefully handling errors (`try`, `except`, `finally`).
- Managing multiple exceptions and creating **Custom Exceptions**.

### 1️⃣2️⃣ Threads & Concurrency ⚡
- Understanding Python's Global Interpreter Lock (GIL).
- **Threading** and **Multiprocessing** to run parallel/concurrent tasks.

### 1️⃣3️⃣ Async Python 🚀
- Writing non-blocking code using `async`/`await`.
- Handling background workers, daemon vs non-daemon threads.
- Recognizing concurrency issues like **Race Conditions** and **Deadlocks**.

### 1️⃣4️⃣ Pydantic
- Data validation and settings management using Python type hints.
- Advanced features and object **Serialization**.
